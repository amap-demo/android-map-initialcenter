# android-map-initialcenter
改变初始地图显示位置

本工程为基于高德地图Android SDK进行封装，实现了改变地图初始显示位置功能。
## 前述 ##
- [高德官网申请Key](http://lbs.amap.com/dev/#/).
- 阅读[参考手册](http://a.amap.com/lbs/static/unzip/Android_Map_Doc/index.html).
- 工程基于Android 3D地图SDK实现
